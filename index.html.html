HTML Basics

Here we will be learning about different html elements and their function while building a web page.

An html element is defined by a start tag "<" with element name and end with close tag "/>"

Elements

Function

html

 This is the root element of any html document that defines the while document

head

 Just like an essay an html file is also divied into head and bodyThis contains the metadata and is placed before body tag and after html tag

title

 This elemnt defines the title of of the html document that appears at the tab bar of the browser

meta

 Althought it making any changes in the interface of the web page. This tag is used by the browser as a guide to display the page or reload it. Another key function of this element is to specifiy the character set, page description, keywords and author of the document, which is used by the search egnines andother web services.

body

 It contains all of the contents of an html documents that is displayed in the web page

p

 This tag is used inside the the body tag. It is used to write a paragraph in the page.

a

 It is used to define an hyperlink in an html document that is used to link one page to another. It is represented by a blue underlined font. When a link is visited it turns purple and if the link is active it turns red.

h

 It is used to format the text in an html file. It starts from h1 till h6 such that h1 is the greatest text size and h6 is the smallest

table

 This is used to create tables in a page. But also can be used to outline the web page without any other language.

tr

 It is used to define a row in a table.

td

 It is used to enter data in a cell of the table

th

 It is used to define the head of a table .

div

 This element is used to define different containers or sections in a page, such that each section could be styled and modified without affecting the other.
